# Prueba React




Se realizo la prueba donde tenemos dos proyectos, una para el front y otra el backend que se utiliza  para login.

- apilogin es donde esta la informacion de usuario, existen dos roles, un administrador y otro externo.
- Desplegamos el backend, ejecutando el comando  "npm start"
-   Admin: "user: acunalema - pass:12345"
-   Externo: "user: fcunalema - pass:123456"
- PruebaKrugerCunalema es el proyecto para el front end, se lo despliega con "npm start"
